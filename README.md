# Full Stack Blog Platform

A full stack blog website built using Node.js, Express.js and MongoDB Atlas.

## 🚀 Features
- User Registration & Login
- JWT Authentication
- Create Blog
- Like Blog
- Image Upload using Multer
- Admin Dashboard
- Dark Mode Toggle

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- HTML
- CSS
- JavaScript

## ⚙️ Run Locally

1. Clone the repository
2. Go to backend folder:
   cd backend
3. Install dependencies:
   npm install
4. Create a .env file and add:

   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key

5. Start the server:
   node server.js

Server runs on:
http://localhost:5000
